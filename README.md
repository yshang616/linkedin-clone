# LinkedIn Clone
## 主要内容：
- 使用CSS和Styled Components还原LinkedIn登录及主页面的布局。
- 实现登录，发表想法，上传图片，分享视频等功能。
- 使用Firebase存储用户信息与部署。

## 技术栈:
 - JS (React) 
 - Redux
 - Styled Components
 - Firebase

## 项目结构:

```txt
└─── be    
    └── public
        └── images       # 图片与Icon素材
    └── src
        ├── actions
        ├── components   
        ├── reducers     
        └── store        # Redux
```

## 下载与使用
```bash
# Clone项目
git clone git@github.com:yshang616/linkedin-clone.git
# 安装dependencies
npm i yarn -g
# 进入项目根目录
cd linkedin-clone
# 运行
npm start
# 应用会在 <http://localhost:3000> 运行
```
